#
[ 创建公会 ]
 1. 需要持有华丽金属G. 
 (华丽金属G可在道具商城中购买.)
 
 2. 在持有华丽金属G的情况下在聊天窗中输入 
 '/公会 (公会名称)'或者
 点击'创建公会'按钮，创建公会.
 
 3.  ALT+G 可显示公会信息栏.
#
 
[ 公会 信息栏 ] 
#
 1. 可以确认公会名称, 经验值, 公会 级别, 公会徽章, 
 公会长名称,公会会员数,上缴点数，会员平均级别, 
 联盟, 敌对 公会的信息
#
 
 2. 会员信息 : 
 可查看个别公会会员的信息,
 右击可管理自己的公会。
 退出公会只有自己可以决定，
 也可被公会长强行踢出公会
#
 
 3. 职位设置 : 
 可设施公会会员的公会经验值上缴和职位。
 上缴最多50%. 
 上缴到公会的经验值在退出公会时 
 初始化.
#
 
 4. 公会技能 : 
 公会等级每提升一级会生成1个技能点， 
 可将此投入到公会技能中
 使用
#
 
 5. 被踢出公会者 : 
 显示被公会强行踢出的玩家目录和
 原因.
#
 
 6. 公会长可通过聊天窗发布公告等
 方便公会会员知晓.
#
 
[ 徽章上传及变更 ]
#
 1) 可使用24 X 24像素 256色 BMP格式 图片
 来制作徽章.
#
 
 2) 在仙境传奇所在的文件夹中创建 Emblem 文件夹 
 将徽章图片复制到该文件夹中.
#
 
 3) 点击公会信息栏显示徽章处的编辑按钮 
 便可应用你所选择的徽章.
#
 
 4) 公会徽章只有公会长才可以进行变更, 
 可随时进行变更
#
